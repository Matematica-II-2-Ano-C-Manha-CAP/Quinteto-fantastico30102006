WILLOW PROJECT

O que é Willow Project?

O projeto Willow é um projeto de perfuração de petróleo da ConocoPhillips localizado na planície do North Slope do Alasca, na Reserva Nacional de Petróleo do Alasca. O projeto era originalmente para construir e operar até cinco plataformas de perfuração para um total de 250 poços de petróleo.

Projeto polêmico de exploração de petróleo nos EUA gera turbulência no governo Biden

O projeto de perfuração de US $ 8 bilhões, é a proposta de criar novos campos de petróleo, o governo americano pode produzir até 180.000 barris de petróleo por dia, 1,5% da produção total de petróleo dos EUA, se aprovado terá um potencial extremamente lucrativo para os Estados Unidos sendo representado como o maior campo de petróleo dos Estados Unidos em décadas.
